# Security Policy

## Supported Versions

Only the latest version of the Smawl SMP Landing Page codebase is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability within this project, please report it immediately to ensure the safety of our player community.

**Do NOT open a public GitHub issue for security-related matters.**

Instead, please send an email with detailed replication steps to the server developer or contact us directly via the official WhatsApp line:
* **Contact Number:** `+62 882-0070-36590` (WhatsApp - Sell Plugin/Dev)

We will investigate all reports promptly and coordinate a fix/patch before public disclosure. Thank you for keeping the Smawl SMP community secure!
